Global IP 確認
================

```sh
brew install curl
curl ipcheck.ieserver.net
curl ipecho.net/plain ; echo
curl -v httpbin.org/ip
```

or

```sh
wget -V
Wget 1.13
wget -q -O - ipcheck.ieserver.net
```
