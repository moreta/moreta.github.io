##### Recent Change Files

`command + shift + e`

#### Navigate

##### Declation

`command + .`

##### Back (file追跡history : custom setting)

`command + [`

##### Forward (file追跡history : custom setting)

`command + ]`

###### Hide All Tool Windows (custom setting)

##### Editor Tabs

`ctrl + 0`
