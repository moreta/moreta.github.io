time
========

# 実行時間を測る

```sh
time sleep 10
       10.00 real         0.00 user         0.00 sys
```

| 項目 | 意味                                                                 |
| ---- | -------------------------------------------------------------------- |
| real | プログラムの呼び出しから終了までにかかった実時間（秒）               |
| user | プログラム自体の処理時間（秒）（ユーザCPU時間）                      |
| sys  | プログラムを処理するために、OSが処理をした時間（秒）（システム時間） |
