## sdk update

`gcloud components update`


## instance list

`gcloud compute instances list`


## install

```
curl https://sdk.cloud.google.com | bash
```

## Set up gcloud

```
gcloud init

> Enter project id you would like to use: googleで作成したprojectのidを入れる
```
