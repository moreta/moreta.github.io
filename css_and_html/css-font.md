## font-awesomeを利用し hoverでfontが変わるようにする。

*scss*を利用している場合

.icon-unlock:hover {
  @extend .icon-lock;
}

## font size


```
body {
  font-size:62.5%;
}
```

この設定をすると 16px = 1.6em同じぐらいになるので、どっちを使ってもいい感じになる
