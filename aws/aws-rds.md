

# DB インスタンスを変更

+ 変更をしてもすぐ反映されるわけではない。
+ 変更をすぐ反映したい場合には ApplyImmediatelyをtrueにする
+ 変更した時の影響は下記のリンクから確認する
  + [DB インスタンスを変更し、パラメータをすぐに適用](htps://docs.aws.amazon.com/ja_jp/AmazonRDS/latest/UserGuide/Overview.DBInstance.Modifying.html)
