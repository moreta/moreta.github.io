

# install python3

```sh
brew install python3
python3 --version
```

# python3でよく使うライブラリをインストール

```
pip3 install numpy  # 線形代数
pip3 install scipy  # 数式処理
pip3 install matplotlib   # 描画 
pip3 install pandas  # データ操作
```
