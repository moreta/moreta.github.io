

# 複数のcommitをまとめてcherry-pickする

```sh
git cherry-pick [cherry-pick の始点となるコミット]..[cherry-pick の終点となるコミット]
git cherry-pick 63ca256a5f806c8b237c3ff5c872debc0be12359..738b8de617497b37bc47c06ce9fa9f676c256b61
```
